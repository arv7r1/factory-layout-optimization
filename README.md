# Manufacturing Layout & Material Flow Optimization

## Overview
This project analyzes a simulated production floor layout to evaluateoperator travel distances and optimize workstation placement using AutoCAD and Excel-based material flow analysis.

## Objective
The objective of this study was to reduce non-value-added operator
movement and improve material flow efficiency by redesigning the
production layout.

## Tools Used
- AutoCAD (2D layout design)
- Excel (material flow analysis)

## Methodology
1. Created an initial factory floor layout in AutoCAD.
2. Mapped operator movement using a spaghetti diagram.
3. Measured travel distance between workstations.
4. Redesigned the layout to align workstations with process flow.
5. Compared operator travel distances before and after layout changes.

## Results

| Layout | Distance per Cycle | Distance per Hour |
|------|------|------|
Before Layout | 90.93 m | 1818.6 m |
After Layout | 24.98 m | 499.6 m |

The optimized layout reduced operator travel distance by **72.5%**.

## Key Insights
- Initial workstation placement caused excessive operator movement.
- Linear workstation sequencing significantly improved material flow.
- Layout optimization reduced unnecessary walking and improved
  operational efficiency.
